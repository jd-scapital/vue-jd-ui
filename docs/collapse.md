# 折叠面板

## 功能

点击下拉展示更多内容

## props

| 参数 | 说明 | 必填 | 类型 | 可选值 | 默认值 |
| --- | --- | --- | --- | --- | --- |
| type | 有无标题 | 否 | String | noTitle(无标题)/title(有标题) | noTitle |
| open | 是否默认展开 | 否 | Boolean | true/false | false |
| title | 标题（当 type 为 title 时） | 否 | String | - | - |

## 默认组件名

`jd-collapse`
